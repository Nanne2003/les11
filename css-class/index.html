Mijn cheatsheat

Hoe verstuur je bestanden via git naar github?

Cd naam van map

Hiermee navigeer je naar de map waar je in wilt werken

Git init

Hiermee maak je een nieuwe repository in de geselecteerde map

Git add naam van bestand

met deze command voeg je de file toe aan de staging area

git commit -m beschrijving

Met deze command maak je het bestand klaar om verstuurt te worden

git branch -m main

met deze command maak je jezelf de branch master en dit is nodig om de commands uit te voeren

Git remote add origin https://github.com/username/naamvanrepository.git

Hiermee link je git aan je github repository

git push -u origin master

Hiermee verstuur je de bestanden naar github
